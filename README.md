# FINDMYASS TOOL
    version 1.0 tested on Mac OS with python 3.9
## ABOUT
Domain informations Gathering such as :
- Subdomains
- (A, AAAA, NS, MX, SOA, TXT) Records
- WHOIS
- Cloudflare detect ...
- Generate a report containing grabbed data
